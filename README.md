## Установка и запуск
<a id="getting_started"></a>

- Клонировать проект

```
git clone https://github.com/GaynetdinovR/YtYt.git
```

- Установить необходимые пакеты

```
npm install
```

- Запустить локальный сервер

```
gulp
```

- Открыть в браузере

```
http://localhost:3000/
```
